ODataExternalEdmModelSample
-----------------

This sample shows how to use an external IEdmModel with Web API 2 OData. This
concrete example exposes the IEdmModel generated by Entity Framework as an OData
service.

This sample is provided as part of the ASP.NET Web Stack sample repository at
http://aspnet.codeplex.com/
