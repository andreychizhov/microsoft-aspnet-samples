ODataAttributeRoutingSample
--------------------

This sample shows how attribute routing works with ODataRouteAttribute.
Attribute routing will specify the routes for the actions in the controller.

The sample illustrates how to route some actions and bind parameters with attribute routing.
1. Add ODataRoutingAttribute on the action with an OData path template as the parameter.
2. The parameter of the action will be binded according to the template.
3. The name of the action and parameters can be arbitrary.
