UploadXDocumentSample
---------------------

This sample illustrates uploading an XDocument to an ApiController using 
PushStreamContent and HttpClient. 

For a detailed description of this sample, please see
http://blogs.msdn.com/b/henrikn/archive/2012/02/17/push-and-pull-streams-using-httpclient.aspx

This sample is provided as part of the ASP.NET Web Stack sample repository at
http://aspnet.codeplex.com/
