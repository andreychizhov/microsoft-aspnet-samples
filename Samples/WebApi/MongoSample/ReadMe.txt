MongoSample
-----------

This sample illustrates how to use MongoDB as the persistent store for an ApiController using 
a repository pattern.

For a detailed description of this sample, please see
http://blogs.msdn.com/b/henrikn/archive/2012/02/19/using-web-api-with-mongodb.aspx

This sample is provided as part of the ASP.NET Web Stack sample repository at
http://aspnet.codeplex.com/
